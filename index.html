Aula 02 de Programação Web II - Programação com servidor web - Apache

Xampp - Gerenciador de arquivos

Localhost = dominio local, é o servidor na máquina

Dashboard = Página index do apache

index.html = configuração do apache para abrir direto um arquivo index, sem a necessidade de clicar em outros arquivos.

Navegador entende apenas duas linguagens = html + js - tem um moto que reconhece apenas js, os demais, abre apenas como arquivos de texto

PHP - é uma linguagem interpretada, serve para transformar os arquivos em html para poder ser entregue ao cliente

PHP - usa tags especiais para sua edição, para a escrita do cógigo em php é utilizado , não aparece nenhuma informação, pois o navegador não entende, mas por estar em .html, ele tenta abrir

echo - em php, serve para mostrar na tela, tal qual printf para a linguagem c

É possível utilizar html e php na mesma página, a diferença é que quando houver tags php, serão interpretadas como php, o restante será como html, sendo assim, um código misto
